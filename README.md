## Структура проекта

### `Math/`
- **`Vector.py`** — класс для работы с векторами:  
  сложение, вычитание, скалярное произведение, норма (∞-норма),  
  чтение/запись из файла и консоли, заполнение случайными числами.

- **`TridiagonalMatrix.py`** — класс для работы с трёхдиагональными матрицами:  
  сложение, вычитание, умножение на вектор, чтение/запись.  

  *Соответствует требованиям:*
  - хранение в виде трёх векторов (`a`, `b`, `c`);
  - индексация элементов **начиная с 1**;
  - запрет на двумерный индексатор;
  - поддержка трёх режимов генерации:
    - `'random'` — случайная матрица;
    - `'dominant'` — диагонально-доминирующая;
    - `'ill_conditioned'` — плохо обусловленная.

### `main.py`
- **`DataGenerator`** — генерирует пару: трёхдиагональная матрица + точное решение **с едиными границами** `[low, high]` для обоих.
- **`ExperimentUtils`** — содержит:
  - `progonka(...)` — устойчивый метод (A);
  - `unstable(...)` — неустойчивый метод (B);
  - `calculate_error(...)` — вычисление абсолютной и относительной погрешностей  
    (с корректной обработкой значений, близких к нулю: используется √ε).

#### Функция эксперимента
`def main_experiment(exp_count: int, low=-1.0, high=1.0, condition_type='random')`
* Проводит exp_count экспериментов для каждого из 6 размеров системы: n ∈ {2, 4, 8, 16, 32, 64} (логарифмическая шкала по основанию 2).
* Генерирует случайную систему указанного типа (condition_type), вычисляет правую часть по известному решению, решает двумя методами и оценивает погрешности.

Возвращает pandas dataframe для последующей записи в csv или xml файл.

*Столбцы:*
* 1 столбец — размер системы (логарифмическая шкала по основанию 2);
* 2 столбец — диапазон заполнения
* 3 столбец — абсолютная погрешность метода прогонки (A);
* 4 столбец — относительная погрешность метода прогонки (A);
* 5 столбец — абсолютная погрешность неустойчивого метода (B);
* 6 столбец — относительная погрешность неустойчивого метода (B).

### make_diagram.py
Скрипт для автоматической генерации диаграмм на основе CSV-файлов:
* Влияние размера системы на погрешность;
* Влияние обусловленности;
* Влияние диапазона заполнения;
* Сравнение методов A и B.

## Использование

2. Запустите plot_results.py — диаграммы появятся в plots/.
