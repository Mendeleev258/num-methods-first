## Структура проекта
### Math/
`TridiagonalMatrix.py` - класс для работы с трёхдиагональными матрицами (сложение, вычитание, умножение на вектор, считывание из файла / с экрана, вывод в файл / на экран)

*Отвечает требованиям:*
* хранение матрицы как трёх векторов;
* индексация элементов начиная с единицы;
* нумерация диагональных векторов по номеру строки;
* запрет на использование двумерного индексатора для матрицы.

`Vector.py` - класс для работы с векторами (сложение, вычитание, скалярное умножение, вычисление нормы как максимальной по модулю компоненты, считывание из файла / с экрана, вывод в файл / на экран, заполнение случайными числами из заданного диапазона)
### main.py
`class DataGenerator` - реализует генерирацию данных для эксперимента: матрицу и вектор х

`class ExperimentUtils` - реализует методы для проведения эксперимента:
* Решение СЛАУ методом прогонки (A)
* Решение СЛАУ неустойчивым методом (B)
* Выичисление абсолютной и относительной ошибки

#### Функция эксперимента
```py
def main_experiment(exp_count: int,
                    matr_low = -1.0, matr_high = 1.0,
                    x_low = -1.0, x_high = 1.0):
```
Получает количество экспериментов, проводимых на матрицах одного размера. Всего рассматривается 10 размеров, задаваемые логарифмической шкалой. На каждый размер приходится 10 экспериментов (передаётся как аргумент в данную функцию). Итого производится 100 экспериментов.

Возвращает pandas dataframe для последующей записи в csv или xml файл.

*Столбцы:*
* 1 столбец — размер системы (логарифмическая шкала по основанию 2 с промежуочными значениями);
* 2 столбец — абсолютная погрешность метода прогонки (A);
* 3 столбец — относительная погрешность метода прогонки (A);
* 4 столбец — абсолютная погрешность неустойчивого метода (B);
* 5 столбец — относительная погрешность неустойчивого метода (B).
* 6 столбец — обусловленность системы, т.е. |A| * |A^-1|

### make_diagram.py
